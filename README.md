# PNL_UNIDAD3

Trabajo de la **Unidad 3** de la asignatura **Procesamiento de Lenguaje Natural (PLN)** de la **Universidad ICESI**.

## Descripción del proyecto

Este repositorio contiene el desarrollo práctico de modelos basados en **Transformers** para el procesamiento de lenguaje natural.  
El trabajo se centra en:

- **Aplicar modelos de Transformers** a un problema real de PLN.
- **Entrenar y evaluar** un modelo a partir de datos textuales.
- **Analizar y comparar resultados**, identificando ventajas y limitaciones del enfoque.

El notebook principal del proyecto es:

- `ultimo_transformers_v1_1_(2).ipynb`

En este notebook se realiza el flujo completo:

1. Carga y exploración de los datos de texto.  
2. Preprocesamiento y preparación de los datos para el modelo.  
3. Definición y configuración del modelo basado en Transformers.  
4. Entrenamiento del modelo.  
5. Evaluación cuantitativa y cualitativa de los resultados.  

## Contexto académico

Este trabajo corresponde a la **entrega de la Unidad 3** del curso de Procesamiento de Lenguaje Natural, donde se busca que el estudiante:

- Comprenda el funcionamiento básico de los modelos de Transformers.
- Sea capaz de implementar una solución usando herramientas modernas de PLN.
- Documente el proceso de experimentación y sus hallazgos.

## Requisitos (vista general)

Para ejecutar el notebook se recomienda contar con:

- Python 3.x  
- Librerías habituales de ciencia de datos y PLN (por ejemplo: `numpy`, `pandas`, `scikit-learn`, librerías de Transformers, entre otras).  

(Los requisitos específicos pueden variar según las versiones utilizadas en el entorno local).
